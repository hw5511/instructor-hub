ComfyUI

ComfyUI를 활용한 AI 이미지 생성의 기초부터 실무 워크플로우까지 배우는 강의입니다. 노드 기반 인터페이스로 Stable Diffusion을 자유롭게 다뤄보세요.

자료

  • 1주차 - ComfyUI 설치 및 기본 세팅
  • 2주차 - 기본 워크플로우 구성
  • 3주차 - ControlNet & IP-Adapter
  • 4주차 - 실전 프로젝트

Q&A

Q ComfyUI 설치가 안 돼요. 어떻게 하나요?
A Python 3.10 이상이 설치되어 있는지 확인해주세요. Windows의 경우 portable 버전을 권장합니다. 1주차 자료의 설치 가이드를 참고하시고, 그래도 안 되면 수업 시간에 질문해주세요.
Q GPU가 없어도 수업을 들을 수 있나요?
A 네, 가능합니다. 수업 중에는 클라우드 GPU 환경을 안내해드리며, CPU로도 기본적인 실습은 가능합니다. 다만 생성 속도가 느릴 수 있습니다.
Q 수업 자료는 언제 올라오나요?
A 각 주차 수업 시작 전까지 이 페이지에 업로드됩니다. 알림이 필요하시면 수업 시간에 말씀해주세요.
Q 커스텀 노드는 어떻게 설치하나요?
A ComfyUI Manager를 먼저 설치하시면 편리합니다. custom_nodes 폴더에 git clone으로 직접 설치하거나, Manager UI에서 검색하여 설치할 수 있습니다.
Q 모델 파일은 어디에 넣어야 하나요?
A models/checkpoints 폴더에 체크포인트(.safetensors)를 넣으시면 됩니다. LoRA는 models/loras, VAE는 models/vae 폴더에 각각 넣어주세요.
Q VRAM이 부족하다는 에러가 나요.
A ComfyUI 실행 시 --lowvram 옵션을 추가해보세요. 이미지 해상도를 512x512로 낮추거나, fp16 모델을 사용하면 VRAM 사용량을 줄일 수 있습니다.
